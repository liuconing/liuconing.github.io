Node.js 學習第一天

在剛開始接觸 Node.js 時，我就有去碰前端金三角 HTML、CSS、JavaScript 雖然來還不是很熟悉，但大致上可以按到簡單的設計版面來去寫 Code 一個靜態式網頁了，所以我來去接觸一些後端東西就是這個 Node.js 看能不能單獨寫出一些常見的網站類型。

這第一天我打算寫如何做 Node.js 環境設定，如果已經安裝好的人可以等待我下一篇文章。

如何安裝 Node.js 環境

首先你要有命令提示字元這個 CMD 工具，當然用 Git 控制器來寫也是可以的這邊我會提供兩種作業系統連結。
Windows Git 去點 Downloads 來下載安裝開啟 Git bash 檔案就好。
MAC iterm2 一樣 Download 下載安裝。
兩種的指令基本上可以共用，還是有一點點小差異就需要去 google 一下了。
