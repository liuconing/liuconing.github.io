Node.js 的 30 天學習日記--第一天

前言

為什麼我要寫這篇學習日記呢，在聽過一些人的建議是說嘗試自己是否能夠堅持不斷學習，所以我來寫這篇 Node.js 的 30 天日記，今天是我的第一天，在邊看 六角學院 的課程中邊學習邊寫筆記與日記，希望我能夠堅持下去。

如果在看的過程中有任何錯誤歡迎寫 E-MAIL： pp840405@gmail.com 提醒我錯誤。

Node.js 學習第一天

在剛開始接觸 Node.js 之前，我就有去學習前端金三角 HTML、CSS、JavaScript 雖然不是說很精通，但大致上可以按照簡單的設計版面來去寫 Code 完成一個靜態式網頁了，所以現在來去接觸後端就是這個 Node.js 看能不能獨自寫出一些常見的網站類型。

這第一天我打算寫如何做 Node.js 環境設定，如果已經安裝好的人可以等待我下一篇文章。

如何安裝 Node.js 環境

安裝 Git 工具

首先你要有命令提示字元這個 CMD 工具，當然用 Git 控制器來寫也是可以的這邊我會提供兩種作業系統連結。

Windows Git 去點 Downloads 來下載安裝開啟 Git bash 檔案就好。

MAC iterm2 一樣 Download 下載安裝。

兩種的指令基本上可以共用，還是有一點點小差異就需要去 google 一下了。

安裝 Node.js

結語
