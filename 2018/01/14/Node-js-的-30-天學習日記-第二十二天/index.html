Postman - API 管理工具

管理 API 好用的工具 Postman ，請按照自己的作業系統版本去安裝。

Postman 介紹

Postman - 取得 get 資料、基礎操作

  1. 請先建立下面 app.js 檔案。

  2. 在 postman 新增一個專案資料夾 nodetest

  3. 新增好啟動 app.js 把 127.0.0.1:3000 你的網址貼到 postman 的 GET 欄上面然後點擊 send 下面欄位就會取得你 JS 檔案送出的資料。
