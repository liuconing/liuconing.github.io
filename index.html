forEach、orderByChild - 資料排序篇

這資料排序的語法是 firebase 的，所以今天學的事如何把 firebase 資料庫裡面的資料由小到大。
orderByChild 的語法是選取要排序的子鍵，單獨使用是沒辦法排序要跟 forEach 使用來完成排序。
先看 firebase 的排序規則
基本上規則是:

  1. 子鍵值 null
  2. 子鍵值 false
  3. 子鍵值 true
  4. 子鍵值是數值由小到大
  5. 子鍵值是字串由 A 到 Z
  6. 物件定義
    以上規則都是由上到下排列

結語
